# GSK DPI Microsite

A modern, password-protected carousel presentation microsite built for GSK Digital Product Innovation.

## 🚀 Features

- **Password Protected**: Secure access with custom authentication interface
- **Responsive Carousel**: Smooth multi-slide navigation with touch-friendly controls
- **Navigation Modal**: Quick access to all slides with visual preview cards
- **Progress Tracking**: Visual progress bar to track presentation progress
- **Modern Design**: Clean, minimalist UI with GSK brand colors
- **Optimized Performance**: Lightweight single-page application
- **Mobile Friendly**: Fully responsive design for all screen sizes

## 📋 Project Structure

```
index.html          # Main application file (HTML, CSS, and JavaScript)
```

## 🎯 Key Components

### Authentication
- Password-protected access with custom modal interface
- Secure entry point before content access

### Carousel System
- Full-screen slide navigation
- Arrow buttons for previous/next transitions
- Keyboard navigation support
- Smooth transitions with optimized animations

### Navigation Features
- Floating action button for navigation modal
- Grid-based slide overview
- Current slide indication
- Quick jump to any slide

### Visual Design
- GSK brand color palette (Orange: #E8490F, Dark: #1A1A1A)
- Custom typography and spacing
- Polished interactive elements
- Accessibility-focused styling

## 🛠️ Development

This is a single-page HTML application. No build tools or dependencies required.

### Local Development
1. Clone the repository
2. Open `index.html` in a web browser
3. Enter the password to access the presentation

### Customization
- Modify styles in the `<style>` tag
- Update slide content in the HTML body
- Adjust password validation logic in the JavaScript section
